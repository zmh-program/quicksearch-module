# quicksearch-module
🔍 Python 模块/全局变量的方法模糊搜索 (在微机课没事干做的算不上小东西的小东西)

### 特性
- √ Python 3.1+ (兼容性好)
- √ 无第三方库依赖 (适合无网环境)
- √ 模糊搜索 
  (编不下去了)
